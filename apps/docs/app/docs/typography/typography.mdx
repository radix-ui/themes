# Typography

Typography system with 10-step scale (0-9), font weights, and formatting components. Kookie UI Typography is extracted from [Radix Themes](https://www.radix-ui.com/themes) with refined values for better readability. 

For complete typography documentation and advanced customization, visit the [Radix Themes typography docs](https://www.radix-ui.com/themes/docs/theme/typography). Note that Kookie UI's typography values may be enhanced and might not match exactly what's shown in the Radix documentation.

</Section>

<Section size="2">

## Usage

Use Text and Heading components to render titles and body copy. These components share `size` and `weight` props for consistent typography.

<CodeBlock>

```tsx
import { Text, Heading } from '@kushagradhawan/kookie-ui';
```

</CodeBlock>

<CodeBlock>

```tsx
<Heading size="4">Page Title</Heading>
<Text size="3">Body text content</Text>
```

</CodeBlock>

</Section>

## Type Scale

10-step size scale (0-9) with corresponding font size, line height, and letter spacing values.

<CodeBlock>

```css
/* Type scale tokens */
var(--font-size-0)  /* 10px - Extra small text */
var(--font-size-1)  /* 12px - Small text */
var(--font-size-2)  /* 14px - Body text */
var(--font-size-3)  /* 16px - Standard text */
var(--font-size-4)  /* 18px - Large text */
var(--font-size-5)  /* 20px - Heading text */
var(--font-size-6)  /* 24px - Large heading */
var(--font-size-7)  /* 28px - Display text */
var(--font-size-8)  /* 35px - Hero text */
var(--font-size-9)  /* 60px - Massive text */
```

</CodeBlock>

## Font Customization

Kookie UI makes font customization simple and flexible. Switch between font families instantly using theme props or customize with your own fonts.

<CodeBlock>

```tsx
/* Sans-serif fonts (default) */
<Theme fontFamily="sans">
  <Text>This uses Inter font</Text>
</Theme>

/* Monospace fonts */
<Theme fontFamily="mono">
  <Text>This uses JetBrains Mono font</Text>
</Theme>

/* Serif fonts */
<Theme fontFamily="serif">
  <Text>This uses Playfair Display font</Text>
</Theme>
```

</CodeBlock>

</Section>

## Custom Fonts

Kookie UI uses a simple two-tier font system. Override the theme variables (`--font-sans`, `--font-mono`, `--font-serif`) to customize fonts globally.

<CodeBlock>

```css
/* Import your preferred fonts */
@import url('https://fonts.googleapis.com/css2?family=Poppins:wght@300;400;500;600;700&display=swap');

/* Override Kookie UI theme font variables */
:root {
  --font-sans: 'Poppins', -apple-system, BlinkMacSystemFont, sans-serif;
  --font-mono: 'Fira Code', 'Consolas', monospace;
  --font-serif: 'Georgia', 'Times New Roman', serif;
}
```

</CodeBlock>

## Enhancements

Kookie UI refines the Radix Themes typography system with improved readability and better visual hierarchy:

- **New Size 0**: Extra small text (10px) for fine print and micro labels. Better granularity for more precise control over text hierarchy and enhanced accessibility for compact interfaces.

- **Enhanced Readability**: Optimized line heights for better spacing and improved reading experience. Refined letter spacing for tighter text flow and improved font weights for more distinct differences.

- **Better Hierarchy**: Clearer size progression where each step provides meaningful visual difference. Consistent spacing for better vertical rhythm across components and enhanced contrast for improved accessibility.
