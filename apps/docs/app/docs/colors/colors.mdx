# Colors

Accessible color system with 12-step scales, automatic dark mode, and P3 gamut support. Kookie UI Colors are extracted from [Radix Colors](https://www.radix-ui.com/colors) with enhanced values and improved accessibility. 

For complete color palette and usage guidelines, visit the [Radix Colors](https://www.radix-ui.com/colors) documentation. Note that Kookie UI's color values may be enhanced and might not match exactly what's shown in the Radix documentation.

## Usage

Colors are controlled through the Theme component's `accentColor` and `grayColor` props. All components automatically inherit these theme colors, but individual components can override with their own `color` prop. `grayColor` also accepts `'auto'`, which matches a neutral gray to the chosen accent.

<CodeBlock>

```tsx
<Theme accentColor="blue" grayColor="slate">
  <YourApp />
</Theme>
```

</CodeBlock>

<CodeBlock>

```tsx
<Theme accentColor="crimson" grayColor="auto">  /* auto picks a matching gray for crimson */
  <YourApp />
</Theme>
```

</CodeBlock>

</Section>

## Component Overrides

Individual components can override theme colors using their `color` prop.

<CodeBlock>

```tsx
<Theme accentColor="blue">
  <Button color="crimson">Override</Button>
</Theme>
```

</CodeBlock>

<CodeBlock>

```tsx
<Theme accentColor="green">
  <Button color="blue">Different Color</Button>
</Theme>
```

</CodeBlock>
